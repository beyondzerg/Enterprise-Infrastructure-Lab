# Enterprise Infrastructure Lab

> Enterprise network infrastructure built using Windows Server 2022, MikroTik RouterOS, Proxmox VE and WireGuard VPN.

---

## 📖 Overview

This repository documents the design, deployment and administration of a small enterprise network infrastructure.

The laboratory environment includes:

- Windows Server 2022
- Active Directory Domain Services
- DNS Server
- DHCP Server
- MikroTik RouterOS
- VLAN segmentation
- WireGuard VPN
- Proxmox VE
- PowerShell automation
- File Server
